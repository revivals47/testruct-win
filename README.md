# Testruct (Windows) — 配布リポジトリ

Testruct Windows 版の**配布専用**リポジトリです。ソースコードは含みません（ソースは非公開リポで管理）。

- `latest.json` — アプリ内「更新を確認」が参照する版マニフェスト（`{version, url, notes}`）
- 各バージョンの **Release** に Windows インストーラ `Testruct-Setup-<ver>.exe` を添付

## ダウンロード（利用者向け）
最新版はこちら → https://github.com/revivals47/testruct-win/releases/latest

ダウンロードした `Testruct-Setup-<ver>.exe` を実行 → 日本語ウィザードでインストール。
未署名のため初回は SmartScreen が出ます：「詳細情報」→「実行」。
